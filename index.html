Methodology
TCE Calculation

1. Compute loadable quantity from DWT, TPC, draft, constants
2. Calculate voyage days (laden + ballast + port times)
3. Gross revenue = freight $/t × lift (mt)
4. Deduct: port DAs, commissions, bunker cost, misc
5. TCE = net revenue ÷ voyage days
6. Report MAX(warranted TCE, eco TCE)

Model Coefficient

For each (freight, bunker) cell in the 5×5 matrix:
Coefficient = Vessel TCE ÷ BCI TCE

Model coefficient = simple average across all 25 cells in the evaluation range for the selected duration bucket.

Constrained: 0 < coefficient < 2. Negative TCE returns excluded.

Fair Value

WA FFA = days-weighted average of monthly BCI 5TC FFA rates over the charter period.

Fair value (fixed) = WA FFA × coefficient
Fair value (index) = coefficient × 5TC BCI

Scrubber: daily premium = effective daily consumption × VLSFO/HSFO spread ÷ voyage days

API Setup Guide

Baltic Exchange FFA & Live Bunker Prices
Step 1 — Baltic Exchange API Key

Log in at api.balticexchange.com with your Baltic member credentials. Click the 🔑 icon (top-right profile menu) to generate or copy your API key.

Step 2 — Create a FFA Feed

In the API portal: Feeds → Add Source → Market Data → FFA
Family: BFADRY (Dry Freight Forward)
Category: Capesize | Type: 5TC monthly contracts
Copy the Feed ID from the URL after creation.

Step 3 — API Endpoint
GET https://api.balticexchange.com/v2/feeds/{feedId}/data
Headers: X-API-Key: YOUR_KEY
// Returns JSON: data[] with date, source_code, value
// Source codes: BCI5TC_M01, BCI5TC_M02... (monthly)
CORS Proxy (required for browser)

Browsers block direct API calls. Run this once on your laptop:

# One-time setup (requires Node.js)
npm install -g cors-anywhere
npx cors-anywhere --port 8080
# Then set proxy URL below to: http://localhost:8080/

Your IT team can also host cors-anywhere permanently (takes ~15 min to set up as an internal service).
